About

Copies is a cross-platform clipboard management solution

  • - Powered by iCloud.
  • - Data stored in user's private cloud container.
  • - Based on apple devices, accessible on all platforms.

Native application

Copies is based on apple devices, it is lightweight, it is easy to use.


Web application

Copies.live is a web service made for Copies, it is also powered by iCloud, and available across platforms with the same iCloud account.


How it works

Enable cloud sync feature in application settings then keep it running, the app will monitor clipboard actions system-wide, it's efficient, it's smart: you can setup filters by keywords or application name. When you made a copy in other application, the content will be collected in the app and synced to copies.live automatically.
